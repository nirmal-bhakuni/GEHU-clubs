ğŸ§© Key Features

1. Homepage

Modern hero section with college branding.

Dynamic sections for upcoming events and featured clubs.

2. Clubs Section

List of all clubs with logo, description, and members.

Individual club page showing gallery and uploaded files (from cloud).

3. Events Section

Grid of all upcoming bootcamps/workshops with date, organizer, and description.

Admin/Club dashboard to create/edit/delete events.

4. Cloud Storage Integration

Use Firebase or Cloudinary to store uploaded photos/files.

Store metadata (URLs, filenames) in MongoDB.

5. Authentication

Admin login (manage all content).

Club login (manage only own events/uploads).

Optional student login (bookmark or register for events).

6. Dashboard

Simple dashboard with event list, upload form, and analytics (total uploads, events).

7. UI/UX

Tailwind CSS + Framer Motion for smooth animations.

Responsive layout (mobile/tablet/desktop).

Optional dark mode toggle.
